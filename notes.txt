UL and OL Tgas 
-------------------------------------
UL --> Unorderd List 
OL --> Ordered List 

Basically the UL and OL tags are used for to make the menus. There are two types menu 
a) horizantal menu 
b) Vertical Menu 

UL:
---------
1. In this tag there is no sequance present. 
2. By default it denotes bulltes.
3. Inside <ul></ul> tags there is list. List can be denoted by <li></li> tags.
4. If you want to change the defualt style than you need one attribute called as type. We can change the type as disc, square, circle.
5. The type attribute is depricated for ul tag. You can handle the type attribute using css, we can use two properties 
    a) list-style-type or list-style
    b) list-style-image --> To add image insted of default type. 


OL
-----------------------
1. In this tag there is sequance present. 
2. By default it denotes number.
3. Inside <ol></ol> tags there is list. List can be denoted by <li></li> tags.
4. If you want to change the defualt style than you need one attribute called as type. We can change the type as number, roman number, alphabets.
5. The type attribute is not depricated for ol tag. You can handle the type attribute using css, we can use two properties 
    a) list-style-type or list-style
    b) list-style-image --> To add image insted of default type.
